BelugaAPI-for-PHP
=================

BelugaAPI for PHP
####インストール方法
ドキュメントルート(~htdocs)直下に設置する

* ~htdocs/index.php
* ~htdocs/module/beluga.php
* ~htdocs/module/functions.php

####サンプルコード
* *statuses/home* **ホーム取得**
* *statuses/update* **投稿**
* *statuses/mentions* **返信取得**
* *statuses/room* **ルーム投稿取得**
* *account/following* **フォローしてるルーム取得**
